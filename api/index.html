Carpark API v1.0 - Documentation

Search

Description
Returns a collection of carpark in JSON format based on the lat and lng.

HTTP request
GET http://localhost/is216/wad2_project/WAD2-Project/api/carpark/search.php?lat=lat&lng=lng
where lat is the lat parameter
and lng is the lng parameter

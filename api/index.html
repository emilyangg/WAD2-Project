Description
Returns a collection of MLBB heroes of a certain class in JSON format.

HTTP request
GET http://localhost/sample_midterm/api/hero/search.php?c=mage
where c is the class parameter, and
mage is the value (e.g. mage, fighter, support, tank, assassin, marksman).
